Sobre o projeto

O Sabor Coletivo é um projeto acadêmico desenvolvido para a disciplina de Programação para Internet. A ideia é simples: criar um espaço onde qualquer pessoa possa compartilhar suas receitas favoritas e descobrir novos pratos.

O site foi construído utilizando apenas HTML5, CSS3 e JavaScript puro, sem o uso de frameworks, focando em estrutura semântica, layout responsivo com Flexbox e interatividade no lado do cliente.

  • ✔ Estrutura HTML semântica
  • ✔ Layout responsivo com Flexbox
  • ✔ Formulário validado com JavaScript
  • ✔ Cards adicionados dinamicamente
